Bioinfo-Chatbot
This chatbot project includes:

1. HTML Structure
2. CSS Glassmorphism UI
3. Chatbox with User & Bot message bubbles
4. Options for message bubble shapes (rounded, soft, iMessage style)
5. Chatbot icon on the top bar
6. JavaScript for sending/receiving messages
7. Predefined questions such as:
   • What is bioinformatics?
   • Scope of bioinformatics
   • Applications and uses
   • General questions for BSc Bioinformatics
